<h2> MongoDB </h2>
<h3> How-to Launch on MACOSX</h3>

<code>
>> mongo
</code>
<br>
<code>
>> show dbs
</code>
<br>
<code>
>> use UCI-Database
</code>
<br>
<code>
>> show collections
</code>
<br>
<code>
>> db.income.find({“age”:{“$gt”:35}}
</code>
<br>
<code>
>> db.income.find({“age”:{“$lt”:20}, “education”:”Some-college”})
</code>
